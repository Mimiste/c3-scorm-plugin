# C3 Scorm Plugin

A Construct 3 plugin to connect an LMS running Scorm 1.2 or Scorm 2004 and get/send statements.
Ported from the Construct 2 version here : [C2 Scorm Plugin](https://github.com/Mimiste/c2-scorm-plugin)
